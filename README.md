# HZLauncher
## HZLauncher is a custom Minecraft Launcher written in Rust using [iced](https://github.com/iced-rs/iced) for the gui.

## Features
- [x] Microsoft login
- [ ] Instance support
- [ ] Modpack support
	- [ ] Curseforge
	- [ ] Modrinth
